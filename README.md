# 📊 Foundation of Data Visualization

A beginner-friendly guide to understanding how to turn raw data into meaningful insights using Python, Seaborn, and Plotly.


## 🎯 What This Repository Covers

This repository is designed as a **foundational workshop on data visualization**, helping you go from raw data to interactive dashboards.

You’ll learn how to:

- Clean and prepare data using Pandas and NumPy  
- Perform exploratory data analysis (EDA)  
- Build visualizations using Seaborn  
- Create interactive charts using Plotly  
- Combine insights into a simple dashboard  

---

## 🧩 Workshop Scenario

You’ve just joined as a data analyst at a 5-star hotel chain.

You are given restaurant data, including:
- Total bill amounts  
- Tips  
- Day and time (Lunch/Dinner)  
- Customer details (size, smoker/non-smoker)  

Your task is to answer:
- Which day generates the highest revenue?  
- What are the peak dining hours?  
- What factors influence higher tips?  


## 🛠️ Tech Stack

- **Python** — Core programming language  
- **Pandas** — Data manipulation (rows & columns / DataFrames)  
- **NumPy** — Numerical computations (arrays)  
- **Seaborn** — Statistical visualization  
- **Plotly** — Interactive dashboards
